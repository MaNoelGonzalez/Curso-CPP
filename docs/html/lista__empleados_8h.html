Descripción detallada

Definición de la estructura ListaEmpleados y las funciones para manejarla.

La estructura ListaEmpleados es una lista enlazada simple que contiene nodos que a su vez contienen objetos de tipo Empleado. Las funciones definidas en este archivo permiten crear, agregar, buscar, modificar, eliminar y mostrar la lista de empleados.
